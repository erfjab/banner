## How to Install Banner

```bash
sudo bash -c "$(curl -sL https://raw.githubusercontent.com/erfjab/banner/master/install.sh)" @ install
```

## Banner Commands

### Block Speed Test Sites
```bash
banner ban
```

### Check Status
```bash
banner status
```

### Remove Blocking
```bash
banner unban
```

### Update Script
```bash
banner update
```

### Uninstall Script
```bash
banner uninstall
```

### Show Version
```bash
banner version
```
